개인정보 처리방침

지역아동센터 학습관리 시스템의 개인정보 처리방침을 다음과 같이 정하고 공개합니다.

최종 업데이트: 2025년 9월 21일

1. 개인정보 처리방침의 의의

지역아동센터는 개인정보보호법을 준수하여 이용자의 개인정보를 보호하고, 이를 위한 가명 시스템을 운영하고 있습니다.

가명 시스템 운영
  • 실명 대신 가명만 사용
  • 서버에는 가명만 저장
  • 실명-가명 매칭표는 오프라인 관리
  • 외부 유출 시에도 개인 식별 불가능
수집하는 정보
  • 가명 (예: 오리너구리, 토끼)
  • 학년 정보
  • 학습 포인트 기록 (8과목)
  • 학습 기록 및 진도
  • 특이사항 (선택적)
예시
실제 이름이 "김철수"인 학생은 시스템에서 "오리너구리"로만 기록되며, "김철수"라는 정보는 시스템에 전혀 저장되지 않습니다.

2. 시스템 보안 현황

기술적 보안 조치
  • 적용 Flask 기반 웹 애플리케이션
  • 적용 Firebase Authentication 인증
  • 적용 Content Security Policy (CSP)
  • 적용 XSS/클릭재킹 방지 헤더
  • 적용 HTTPS 강제
접근 제어 및 세션 보안
  • 세션 자동 만료 30분
  • 로그인 실패 시 IP 차단 5회
  • 차단 해제 시간 30분
  • 권한 기반 접근 제어 역할별
  • 쿠키 보안 설정 HttpOnly

3. 데이터 관리 및 백업

백업 시스템
  • • 자동 백업: 매일 22시
  • • 백업 형식: JSON, Excel
  • • 저장 위치: 로컬 파일 시스템
  • • 실시간: 포인트 입력 시 자동 백업
데이터 보관 및 삭제
  • • 보관 기간: 센터 정책에 따라 결정
  • • 삭제 권한: 개발자만 가능
  • • 삭제 이력: 백업에 기록 보관
  • • 복구: 백업을 통한 데이터 복원 가능
